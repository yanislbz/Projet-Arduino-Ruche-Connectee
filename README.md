# Projet-Arduino-Ruche-Connect-e
Projet de prototypage rapide réalisé en une semaine lors de ma 2e année à Polytech Marseille. L'objectif était de concevoir un système IoT complet pour le monitoring de ruches, intégrant la sécurité environnementale (détection d'incendie) et le suivi d'activité biologique (comptage laser), le tout articulé autour d'une architecture Arduino.
